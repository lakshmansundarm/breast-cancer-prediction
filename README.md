# Breast Cancer Prediction

## Overview

A Machine Learning web application built using Streamlit that predicts whether a breast tumor is Benign or Malignant.

## Features

- Upload breast cancer dataset
- Exploratory Data Analysis (EDA)
- Correlation Heatmap
- Multiple Machine Learning Models
  - Logistic Regression
  - SVM
  - Random Forest
  - KNN
  - Decision Tree
  - XGBoost
- Single Prediction
- Batch Prediction
- Download Prediction Results

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Seaborn

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
